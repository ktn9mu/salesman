I use Simulated Annealing with Metropolis acceptance.
A state is an ordered list of cities representing a closed tour (including the return from the last city to the first).

At each step, a trial tour is generated by randomly modifying the current tour using two move types:

Random swap (probability = swapProb, default 0.05): two random cities are swapped to introduce non-local changes and help escape local minima.

Random segment reversal (probability = 1 − swapProb, default 0.95): a randomly chosen city segment is reversed (2-opt style) to efficiently remove path crossings.

The change in distance is computed as DeltaE = L_trial − L_current.
The trial is accepted if DeltaE ≤ 0, otherwise with probability P = exp(−DeltaE / T).

The initial temperature T0 is chosen automatically by sampling many trial moves, taking the largest uphill DeltaE_max, and setting T0 such that
exp(−DeltaE_max / T0) ≈ p,with p ≈ 0.8.

* datareader.cpp : example code to read in the data files (use Makefile)
